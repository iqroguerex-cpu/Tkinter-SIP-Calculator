## ğŸª„ Overview

This SIP Calculator allows users to estimate the future value of their monthly investments using compound interest.

It uses the standard SIP formula:

\[
FV = P \times \frac{(1 + r)^n - 1}{r} \times (1 + r)
\]

Where:

- **P** = Monthly investment  
- **r** = Monthly return rate  
- **n** = Total number of months  

The result is displayed in **Indian number format** (e.g., â‚¹10,00,000.00).
